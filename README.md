
### npm源设置
npm set registry http://npm.corp.rs.com/

## 使用说明:
npm run start: 运行本地开发环境

npm run dev: 开发环境编译

npm run uat: 测试环境编译

npm run stg: 预上线环境编译

npm run prd: 生产环境编译

npm run eslint: 校验代码质量

###目录说明
api nodejs服务器, mock数据用
src 源代码
test 测试代码


### 启动api服务器
npm run api
